Projects — Details

APM For Monitoring Viewership status

Developed the people-meter system that captures HDMI output and sends viewership data to the cloud using an onboard GSM module. I debugged and fixed the MCU SPI handler, implemented an interrupt-driven SPI solution, and developed a custom MCU bootloader supporting OTA from a Raspberry Pi with rollback handling. Skills gained: embedded C, SPI debugging, bootloader development, OTA mechanisms, GSM integration.

Industrial IoT Gateway

Built a gateway supporting MODBUS over RS485 and MODBUS-TCP, with GSM and Ethernet fallback to reliably forward sensor data to AWS for analytics. I designed the project plan, collaborated on hardware bring-up, and implemented drivers and stacks including LwIP, USB, FatFS on STM32H5 series. Skills gained: MODBUS, RS485, TCP/IP (LwIP), STM32 driver development, cloud integration (AWS).

BridgEth Controller

Designed a compact industrial controller using Raspberry Pi CM4 and RP2040, implementing state machines and reliable communication over SPE, Ethernet and MQTT. I developed the FreeRTOS-based framework, scheduling, and IO handling for sensors and actuators, and ensured fault-tolerant DIN rail communications. Skills gained: FreeRTOS, state machine design, MQTT, real-time systems.

Power Optimization on Raspberry Pi (CM4)

Optimized power by disabling unused hardware/software stacks, editing device trees, and building an optimized kernel to reduce consumption. I removed network/WLAN/USB stacks, tuned kernel configs, and measured power savings using Type-C testers. Skills gained: Linux kernel configuration, device tree modification, power management, measurement and validation.

Sensor Fusion & Yocto/Peripheral Integrations

Integrated radar and camera data fusion and developed Yocto images and STM32 sensor drivers (ADXL345), enabling reliable sensor calibration and data parsing. Work included Yocto recipe creation, calibration for camera-radar alignment, and accelerometer I2C integration producing JSON-formatted telemetry. Skills gained: Yocto, sensor fusion, I2C/STM32, data calibration, build systems.

USB2Ethernet using Customized PICO Board

Developed a USB-to-Ethernet solution using a customized Raspberry Pi Pico paired with a LAN865x Ethernet PHY to enable communication between two hosts over Single-Pair Ethernet (SPE). I initialized and driven the LAN865x over SPI, implemented data paths on the RP2040, and validated node-to-node communication, including RNDIS-style host links. Skills gained: RP2040/PICO, SPI driver work, LAN865x, Single-Pair Ethernet (SPE), embedded C.
